This folder contains a script which represent the content of the paper.

The figures showing the sequences of the junctions and the probe sets can be obtained with the script DrawingSequences.R.
The required data sets are hta1.r1.ass, hta1.r1.psr_ps.bed and TC_PSR_JUCID_NumberID.txt which is a mapping file between id's
TC_PSR_JUCID_NumberID.txt is included and the other files can be obtained upon request.
